## 不精密採点DX
ブラウザ上で単音の精度採点を行うWebアプリケーションです...になる予定です。製作中。  

https://03wassan.github.io/fuseimitsu-saiten/

### 使い方
1. リンクを開きます。
2. 「マイクの使用を求めています」というダイアログが出るので「許可」をクリックします。
3. 10秒程度に一回勝手に採点結果が出ます。現状A(ラ)の音だけ採点します。

### 今後製作予定の機能
* ユーザー名入力
* 選曲(音)機能
* UIの改善・画面追加
